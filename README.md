# Eat Da Burger!
​
#### burger, 2/23/20
​
#### By Michael Bradford
​
## Description
A simple exercise in building a full stack application for tracking burger consumption.

## Features
​
* Full stack application.
* Uses MVC model.
* Users can view burgers stored in a database, add burgers, and mark burgers as eaten.
​
​
